# CodeFuson_web

## Run project
Run Django:
`python manage.py runserver `

Run React:
`npm start`

Generate requirements:

    `cd backend` 
    `pip freeze > requirements.txt`

Build docker:
`docker-compose up --build` or `docker compose up --build`

Run docker:
`docker-compose up` or `docker compose up --build`
